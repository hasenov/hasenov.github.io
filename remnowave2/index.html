FAQ — вопросы
и ответы

Вы нажимаете «Установить ключ». Сервис обращается к Remnowave API и возвращает уникальный VLESS‑URL, который активен 30 минут. Добавьте его в клиент (Shadowrocket, V2RayN и т. п.) — и сразу подключайтесь.

Доступ автоматически прекращается. Для нового подключения получите новый временный ключ. Продлёнки и автопродления нет по дизайну сервиса.

Технические логи для поддержания работоспособности минимальны. Трафик‑логи не сохраняются — политика приватности публикуется отдельно (текст уточняется).

iOS: Shadowrocket, Streisand, V2Box. Android: v2rayNG, Nekobox. Desktop: V2RayN (Windows), V2RayU (macOS), Qv2ray (Linux). Список будет расширяться (текст уточняется).
